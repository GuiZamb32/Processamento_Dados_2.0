CONTEXTUALIZAÇÃO
A cesta básica é composta por um conjunto de alimentos considerados essenciais para a alimentação
do brasileiro. Com a composição regulamentada por lei, o preço da cesta básica é frequentemente
utilizado como métrica de prosperidade, dominando a análise do custo de vida no país.
Como parte fundamental do orçamento brasileiro, os itens da cesta básica compõem parte significativa
do índice de preços ao consumidor amplo(IPCA), medida governamental utilizada para compor
indicadores de inflação. A variação desse índice é registrada em uma série histórica do banco central,
disponível para consulta online via API de dados abertos.
Obtendo a série histórica do IPCA, e uma precificação atualizada dos itens da cesta básica, podemos
simular as condições de precificação desses itens no passado. Esse processo é similar ao cálculo do
Preço Real Corrigido, onde calculamos quanto um item deveria custar baseado apenas na inflação.


DESAFIO
Construa uma base de dados e um pipeline de scripts python para extrair e armazenar os dados
necessários para a análise e processamento dos dados do IPCA e dos produtos que compõem a cesta
básica. Este pipeline deve buscar a série histórica do IPCA na API do Banco Central e utilizar Web
Scraping para obter a informação dos produtos da cesta básica de um mercado de Florianópolis.
Utilizando os dados obtidos, extraia relatórios mostrando:

1. O valor e a composição da cesta básica de menor valor no mercado escolhido, nos dias atuais;
    a. Dica: liste, para cada componente da cesta básica, o nome e marca do produto; o preço
    unitário; o número de unidades para atingir o quantitativo pedido; outras informações
    que considerar relevante;.

2. O valor e a composição da cesta básica de maior valor no mercado escolhido, nos dias atuais;

3. O valor estimado para as cestas montadas nos anos anteriores, calculado com base no IPCA
anual acumulado.
    a. Dica 1: você precisará calcular o índice acumulado. Lembre-se que o índice IPCA é em
    relação ao mês anterior, portanto, usa a lógica do juros sobre juros.
    b. Dica 2: estamos usando como base o preço atual dos produtos. Portanto a estimativa
    do valor em anos anteriores deve usar a métrica de deflação.



Para este desafio, considerem uma cesta básica montada com os itens abaixo:
    
    1. Arroz (branco, integral ou parboilizado): 5kg;
    2. Feijão: 2kg;
    3. Óleo de soja: 900ml;
    4. Açúcar: 1kg;
    5. Café: 500g;
    
    
Como bônus, gere um complemento incluindo os itens abaixo. Nos relatórios, o complemento deve ser
mostrado como um campo adicional (mostre o valor da cesta básica e da cesta + complemento)
    1. Macarrão: 1kg;
    2. Farinha de milho ou mandioca: 500g;
    3. Sal: 1kg;
